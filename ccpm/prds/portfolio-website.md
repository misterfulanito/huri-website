# Wikipedia-Style Portfolio Website

## Overview
Single-page portfolio website mimicking Wikipedia's design language while showcasing professional work, experience, and personal brand.

## Target Audience
- Potential employers and clients
- Professional network connections
- Fellow designers and developers

## Technical Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** CSS Modules with CSS Custom Properties
- **CMS:** Tina CMS
- **Media:** Cloudinary
- **Deployment:** Vercel
