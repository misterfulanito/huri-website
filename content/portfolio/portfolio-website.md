# Wikipedia-Style Portfolio

This portfolio website showcases my work using a clean, Wikipedia-inspired design language.

## Features

- **Three-Column Layout**: Sidebar navigation, main content area, and accessibility controls
- **Accessibility First**: Text size controls, width options, and dark mode
- **Content Management**: Blog and portfolio managed via Tina CMS
- **Responsive Design**: Works beautifully on desktop, tablet, and mobile

## Technical Highlights

- Built with Next.js 15 and React 19
- TypeScript for type safety
- CSS Modules with CSS Custom Properties for theming
- Cloudinary for optimized image delivery
- Deployed on Vercel with automatic deployments
